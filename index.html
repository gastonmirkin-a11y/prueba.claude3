⚙️ Falta conectar la base de datos compartida

Para que los cinco integrantes de la escribanía vean la misma información, la página necesita conectarse a una base de datos gratuita de Google (Firebase). Es una configuración que se hace una sola vez:

  1. Entrá a console.firebase.google.com con tu cuenta de Google y creá un proyecto (podés desactivar Google Analytics).
  2. En Compilación → Authentication: botón «Comenzar», habilitá Correo electrónico/contraseña y en la pestaña «Users» creá un usuario: un email (puede ser inventado, ej. escribania@miestudio.com) y la clave compartida que van a usar los cinco.
  3. En Compilación → Firestore Database: creá la base de datos (modo producción, ubicación southamerica-east1). Luego, en la pestaña Reglas, reemplazá todo por:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read, write: if request.auth != null;
        }
      }
    }
    y publicá.
  4. En Configuración del proyecto (⚙️) → Tus apps: agregá una app Web (</>), ponele un nombre y copiá el bloque firebaseConfig que te muestra.
  5. Pegá ese bloque acá abajo y tocá «Guardar configuración».

🔐 Ingreso a la escribanía

Escribí la clave compartida de la escribanía. Queda guardada en este navegador: no hace falta volver a escribirla cada vez.

Agregar escritura

La fecha de pedido se puede dejar vacía y completarla más adelante, directamente en la tabla, cuando pidas los certificados. Todo lo que se carga acá lo ven los cinco integrantes al instante.
