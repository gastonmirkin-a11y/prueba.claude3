⚙️ Falta conectar la base de datos compartida

Para que los cinco integrantes de la escribanía vean la misma información, la página necesita conectarse a una base de datos gratuita de Google (Firebase). Es una configuración que se hace una sola vez:

  1. Entrá a console.firebase.google.com con tu cuenta de Google y creá un proyecto (podés desactivar Google Analytics).
  2. En Authentication: habilitá Correo electrónico/contraseña y en «Users» creá un usuario: un email (puede ser inventado) y la clave compartida que van a usar los cinco.
  3. En Firestore Database: creá la base de datos (modo producción, ubicación southamerica-east1). Luego, en la pestaña Reglas, reemplazá todo por:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read, write: if request.auth != null;
        }
      }
    }
    y publicá.
  4. En Configuración del proyecto → Tus apps: agregá una app Web (</>) y copiá el bloque firebaseConfig que te muestra.
  5. Pegá ese bloque acá abajo y tocá «Guardar configuración».

🔐 Ingreso a la escribanía

Escribí la clave compartida de la escribanía. Queda guardada en este navegador: no hace falta volver a escribirla cada vez.

Agregar escritura a inscribir

El plazo para inscribir es de 45 días corridos contados desde el día siguiente al otorgamiento. Completá la fecha de presentación y el N° de entrada cuando la presentes en el Registro; ahí la escritura sale de las alertas.

Agregar trabajo

La fila se pinta de amarillo cuando pasan 15 días desde la factura sin cobrar, y de rojo a los 30 días. Cargá el monto cobrado (total o parcial) y la fecha de cobro directamente en la tabla; cuando el cobro está completo, el trabajo queda como «Cobrado» y sale de las alertas.
